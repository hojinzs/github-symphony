Implementation in packages/cli/src/ansi.ts:

export function stripAnsi(s: string): string {
  // eslint-disable-next-line no-control-regex
  return s.replace(new RegExp(`${ESC}\\[[0-9;]*m`, "g"), "");
}

Key Points:
1. Uses RegExp constructor to avoid TypeScript control character warnings
2. Dynamically constructs pattern using ESC constant: "\x1b["
3. Pattern matches ANSI escape sequences: \x1b[...m
4. Global flag (g) removes all occurrences
5. ESLint directive suppresses no-control-regex warning
